Coding Experience

Coding at first is a great feeling just like when a small baby starts walking.The feeling of when you write, run and compile your first 'Hello world' program with success cannot be described. For me I first started with Python programming which was very easy at first but after joining the Computer Science community I was advised to start with the low level programming language C Programming. I found it fun as it laid a good foundation in my Coding experience. Furthermore, research has it that most of the current programming languages were invented from C and I tell you learning C is like learning most programming languages since the syntax is almost similar to that of C. I then move up to Learning C++ which is a superset of C. The main reason of learning this language was to introduce one to the concept of Object Oriented Programming which forms the basis of real world applications.After learning this two languages I found it easier to be introduce to other programming languages.

What about Web Development ?

Some of you might be wondering why I am talking all fo this information instead of talking about web development.Don't worry I have got you covered! Web development is currently what live with day and night. I started this journey with a quench of thirst of building websites for people but the journey has not been a linear progress. You must first know how the internet works and a bit of its history. After knowing this then you are supposed to learn HTML5 which is responsible for the markup or content of the webpages. After learning the basics you can move on to CSS which is responsible for the styling and presentation of webpages. To have more fun one has to learn Javascript which enables interactivity with end users in websites. For now this is the journey that one has to take in order to have the asic skills of web developmet. After this is when one can decide a clear of which they can take on their career in web development by either becoming a front end developer, back end developer or a fully fleged fullstack developer.
